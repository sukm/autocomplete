## Requirements

- Node 6.13.0
- Docker

## Quick Start

1. Clone the repo

```
$ git clone git@github.com:sukm/autocomplete.git
$ cd autocomplete
```

2. Install the dependencies:

```
$ npm install
```

3. Run the development server:

```
$ npm start
```
